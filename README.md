# 💼 Job Tracking Application

A full-stack Django web application designed to help users manage and track their job applications efficiently.

This project allows users to organize job applications, monitor application status, store company details, and manage their job search workflow from a single dashboard.

The application was built to practice real-world Django development concepts including authentication, CRUD operations, deployment, database handling, and responsive UI development.
